Step 1. Configure connection

Messages

What to do:

  1. If you are using iPhone - Set the site on your home screen: click "Share" → "Add to Home Screen",
  2. Do it the same on other iPhone or device - they will be Client A and Client B,
  3. If you are using local machine (PC, laptop) - Open this page in browser in two tabs (they will be Client A and Client B),
  4. Client A (first tab or device): Click "Create offer" button,
  5. Client A: Copy the created "offer",
  6. Client B (second tab or device): Paste it into the "offer" input over "Process" button,
  7. Client B: Click "Process" button. In the frame under "Create offer" button will appear "answer",
  8. Client B: Copy the created "answer" from the frame under "Create offer" button,
  9. Client A: Paste it into the "offer" input over "Process" button,
  10. Client A: Click "Process" button,
  11. If everything OK you will see in 'Messages' first message: 'Connection established!',
  12. Start sending messages (Step 2).
